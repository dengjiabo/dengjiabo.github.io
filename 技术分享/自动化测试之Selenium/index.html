1、Selenium简介

Selenium是一套开源软件测试自动化工具，支持所有主要操作系统和浏览器。

1.1、Selenium 内容

（1）、Selenium WebDriver

WebDriver是基于selenium设计的操作浏览器的一套API，针对多种编程语言都实现了这套API。

（2）、Selenium IDE

通过在浏览器中安装该插件，可以完成脚本的录制。

（3）、Selenium Grid

Selenium Grid是一种自动化的测试辅助工具，Grid通过利用现有的计算机基础设施，能加快Web-app的功能测试。利用Grid，可以很方便地同时在多台机器上和异构环境中并行运行多个测试事例。

1.2、优缺点

优点：

（1）、免费
（2）、持续交付工作流程集成
（3）、广泛的支持语言，平台和浏览器
（4）、庞大的社区
（5）、大型插件库

缺点：

（1）、陡峭的学习曲线 --需要掌握一门语言
（2）、没有内置的图像比较
（3）、没有报告功能
